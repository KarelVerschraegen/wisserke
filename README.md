# Wisserke

> A GitHub App built with [Probot](https://github.com/probot/probot) that that automatically adds a "Het eiland" gif if your pull request has more deletions than additions.

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start
```
